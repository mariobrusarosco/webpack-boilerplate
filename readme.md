## Features

* Pure CSS
* Stylus
* Fonts (hosted by you, not by Google Fonts)
* Hot Module Replacement
* Tree Shaking
* Babel
* React Support
* Code Splitting usign 'import()'
* Generate assets as 'gzip' and 'brotli' files (.gz & .br)
* Prettier Support - Ran after every commit
